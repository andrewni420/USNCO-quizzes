PDF scraper for past USNCO multiple choice exams.

Locations.java - Extracts relevant locations of questions and answers into a .txt file

CropPDF.java - Uses the .txt file of locations to cut up a PDF into question and answer .png files

AddTransparency.java - Converts the white background of answers into transparent backgrounds. This is necessary so that the website can display 
green/red backgrounds for right/wrong answers
